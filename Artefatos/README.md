# Artefatos do projeto

Este diretório mantém os artefatos do projeto.

```console
├── diagramas/
│   ├── comunicação/
│   ├── sequencia/
│   ├── sequencia_sistema/
│   ├── casos_de_uso.png
│   ├── classes.png
├── interfaces/
├── memoriais/
├── DocumentacaoDeProjeto.pdf
├── DocumentoDeVisao.pdf
└── README.md
```

- `/diagramas/comunicação/`: Diagramas de comunicação.
- `/diagramas/sequencia/`: Diagramas de sequência.
- `/diagramas/sequencia_sistema/`: Diagramas de Sequências focados apenas na interação do sistema com o usuário.
- `/diagramas/casos_de_uso.png`: Casos de uso do sistema.
- `/diagramas/classes.png`: Diagrama de classes do sistema.
- `/interfaces/`: Protótipos da interface do usuário.
- `/memoriais/`: Memoriais Descritivo da Graduação em Engenharia de Software.
- `DocumentacaoDeProjeto.pdf`: Documento completo com especificações e artefatos do projeto.
- `DocumentoDeVisao.pdf`: Documento que descreve os objetivos e escopo do projeto.
- `README.md`: Este arquivo.
